Вопросы по работе №20

При редактировании объектов в Autoсad обычно выбираются несколько объектов, формирующие ... объектов

Для того чтобы убедиться, что две горизонтальные линии пересекают вертикальную, можно использовать параметр «...» объектной привязки.

С помощью ... можно соединить два объекта, используя дугу с заданным радиусом, касательную к объектам
